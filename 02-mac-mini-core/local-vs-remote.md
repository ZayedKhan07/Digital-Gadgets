# Local vs Remote Mac Access

## Priority: Local First

| Mode | When | Priority |
|------|------|----------|
| Local (desk) | At my workspace with monitor/keyboard | **Primary** |
| Remote (AnyDesk) | Away from desk, traveling, backup | Fallback only |

## Local Setup Requirements

- Mac mini connected to monitor at my desk
- Keyboard + mouse (can be shared with Windows laptop via KVM or just dedicated)
- Wired ethernet to Mac mini
- Mac mini always on, ready to use

## Remote Setup Requirements

- AnyDesk installed on Mac mini
- AnyDesk installed on Windows laptops
- Internet stable enough for remote session (see 01-internet-stability)
- Unattended access configured
