# Mac mini M4 Setup Checklist

## Hardware Setup

- [ ] Mac mini placed at desk
- [ ] Power connected
- [ ] Ethernet cable connected (not WiFi)
- [ ] Monitor connected (HDMI or USB-C)
- [ ] Keyboard connected
- [ ] Mouse/trackpad connected

## Initial macOS Setup

- [ ] macOS updated to latest
- [ ] Apple ID signed in
- [ ] FileVault enabled (disk encryption)
- [ ] Firewall enabled
- [ ] Auto-login configured (for unattended access)

## Dev Environment

- [ ] Homebrew installed
- [ ] Git configured
- [ ] VS Code / cursor installed
- [ ] Claude Code installed
- [ ] Node.js / Python / whatever I need
- [ ] SSH keys generated and added to GitHub

## Remote Access

- [ ] AnyDesk installed
- [ ] Unattended access password set
- [ ] Tested remote connection from Windows laptop

## Backup

- [ ] Time Machine configured (if external drive available)
- [ ] Critical files in Git or cloud storage
